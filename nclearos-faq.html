1. Download the NclearOS or NclearOS2 .iso file from github
2. Launch VirtualBox.
3. Proceed with steps provided on this site
4. Start virtual machine.
If you want OS to work with files create new .vhd Virtual Hard Disk file using Virtual Media Manager in VirtualBox. Then you need to format it using FAT32 filesystem - one way to do it is by using Windows built-in Disk Management: from top left corner select Action>Attach VHD. Choose VHD file created by VirtualBox. Initialize it by selecting MBR and add new simple volume. Detach VHD, run VM and format new disk in NclearOS.

NclearOS is completely open source - using the source code in own projects is allowed. Credit is not required, especially when only a few chunks of code are used. However, when your OS relies heavily on NclearOS code / is only a modified version, a "Based on NclearOS" credit would be appreciated.

To edit the source code:
1. Download NclearOS or NclearOS2 source code.
2. Create new CosmosOS project in Visual Studio.
3. Extract files from .zip to %USERPROFILE%\source\repos\(your project name)

If you want to publish your OS on github, I recommend creating a fork.
